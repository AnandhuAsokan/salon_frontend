# salon_frontend

salon_frontend

Manually type http://localhost:5173/adminsignup to access the admin signup page.
Manually type http://localhost:5173/adminlogin to access the admin login page.

http://localhost:5173/login is the user login page.
http://localhost:5173/ is the user signup page.

------------------------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------------------------
admin section
------------------------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------------------------

After signing up and logging in as an admin, you can create services.
After creating one or more services, go to the staff section, where you can create multiple staff members and select the services that each staff member can perform.

You can change the weekly off days from the staff section. By default, it is set to every Sunday. You can change it at any time, and it will replace the existing weekly off days.

You can add holidays, and there will be no slots available for any staff on those days.

You can change the shop opening and closing times. Available slots will always be between these timings. You can update them at any time.

You can change the status of bookings, staff, and services at any time.

You can view a minimal report in the report section.
-----------------------------------------------------------------------------------------------------
-----------------------------------------------------------------------------------------------------

user section

------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------

The landing page is the user signup page, where users can register and log in.

Services added by the admin will be listed on the home page.

Select a service and choose a date. If the selected date is a holiday, weekly off day, has no available slots, or no staff is available for that service, then slots will not be shown. Otherwise, select a slot from the desired staff member and confirm the booking.

By clicking the user icon at the top-right of the page, you can view your booking history, cancel pending bookings, and log out.
